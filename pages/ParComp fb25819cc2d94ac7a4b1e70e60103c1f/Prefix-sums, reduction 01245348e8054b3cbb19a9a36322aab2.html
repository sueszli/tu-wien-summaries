1) Recursive

Recursive solution

Sum pairwise, then recurse on smaller problems.

Scan(x,n) {
	if (n==1) {
		return;
	}

	for (i=0; i<n/2; i++) {
		y[i] = x[2*i] + x[2*i+1];
	}

	// solve recursively in y -> implicit (with fork-join) or explicit barrier
	Scan(y,n/2);

	// take back
	x[1] = y[0];
	for (i=1; i<n/2; i++) {
		x[2*i] = y[i-1] + x[2*i];
		x[2*i+1] = y[i];
	}

	if (odd(n)) {
		x[n-1] = y[n/2-1] + x[n-1];
	}
}
