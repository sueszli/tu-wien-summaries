Stencil computation

( n × m ) (n \times m) ( n × m ) ﻿ matrix A A A ﻿

Output:

Updated version of A A A ﻿ such that a convergence criteria is met

Matrix borders must be handled in a suitable way

iterate {
	for all (i, j) {
		A[i,j] <- (A[i-1,j] + A[i+1,j] + A[i,j-1] + A[i,j+1])/4
	}
} until (convergence)
