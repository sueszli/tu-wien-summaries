Matrix-Matrix multiplication

Matrix-Matrix multiplication

Calculate with nested parallelism

Input:

(n×l)\small (n \times l)(n×l)﻿ matrix A\small AA﻿

(l×m)\small (l \times m)(l×m)﻿ matrix B\small BB﻿

Output:

(n×m)\small (n \times m)(n×m)﻿ matrix C\small CC﻿

C=A⋅B\small C = A \cdot BC=A⋅B﻿

C[i,j]=∑A[i,k]⋅B[k,j]\small C[i, j]=\sum A[i, k] \cdot B[k, j]C[i,j]= ∑ A[i,k]⋅B[k,j]﻿

par (0<=i<n) {
par (0<=j<m) {
C[i,j] = 0;
for (k=0; k<l; k++) {
C[i,j] += A[i,k]*B[k,j];
}
}
}

On a CREW PRAM, the multiplication can be done in O(l)\small O(l)O(l)﻿ time steps and O(nml)\small O(nml)O(nml)﻿ operations in total, using n⋅m\small n\cdot mn⋅m﻿ processors.
